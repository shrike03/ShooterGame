# ShooterGame
  A game where you play as a soldier and shoot moving targets. 
  When you shoot down a shield, a new one will appear in its place with increased speed, 
  you will also benefit one point.
# Control
  W - jump
  A - moving left
  D - moving right
  LPM - shoot
  R - reset game
  P - pause game
# The purpose of the project
  Presentation of knowledge of the basics of programming in Python. 
  Creating classes and using them in the game code.
